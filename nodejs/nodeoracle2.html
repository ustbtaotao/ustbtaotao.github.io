node-oracle安装 补充说明

0

说明

上一篇我们介绍了oracledb 的安装方法，本篇补充说明先因为oracledb版本升级带来的变更：
1、oracle数据版本10g以下不再支持，ORA-03134: Connections to this server version are no longer supported；
2、如果用pm2 启动包含oracledb驱动的项目，需要使用单进程fork模式，用cluster模式驱动会报错
3、对于同一版本的centeros 如果部署成功过一台，直接部分下oracledb node 包，下次同样的环境，直接安装oracleclient和配置环境变量即可。
